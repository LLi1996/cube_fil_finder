Installing Fil3D
================

.. toctree::

Installation
------------

Fil3D can `eventually` be installed via pip:

.. code-block:: shell

    pip install fil3d


Or you can install from this repo by:

.. code-block:: shell

    python setup.py install


Requirements
------------

Requires:

    * astropy
    * matplotlib
    * numpy
    * scipy

Optional:

    * filfinder (`docs <https://fil-finder.readthedocs.io/en/latest/>`_)
