Installing Fil3D¶

Installation¶

Fil3D can eventually be installed via pip:

pip install fil3d

Or you can install from this repo by:

python setup.py install

Requirements¶

Requires:

  • astropy

  • matplotlib

  • numpy

  • scipy

Optional:

  • filfinder (docs)
